# OpenClaw Directory

A beach-themed directory showcasing creative use cases for AI agents, featuring ideas crowdsourced from the developer community.

## About

This project displays practical and creative applications of OpenClaw AI agents, from morning routines to YouTube recaps to learning path builders.

## Features

- 12+ real-world AI agent use cases
- Clean, responsive design with beach/lobster aesthetic
- Ideas sourced from developer community feedback

## Design

- **Fonts**: Instrument Serif (headers) + Work Sans (body)
- **Theme**: Sandy beach colors with coral accents
- **Framework**: Vanilla HTML/CSS
